# OpenLoops

**A working-memory prosthetic for macOS — for people who lose the thread.**

You start booking a flight, get pulled into Slack, and the booking quietly dies.
You type three sentences of a reply, a meeting starts, the draft is never sent.
Todo apps don't help, because the task is lost *before* you ever write it down.

OpenLoops ambiently observes your activity, infers the *intent* behind it, detects
when a coherent task is abandoned before completion, and captures it as a structured
**Open Loop** — with the URLs, the draft, the price you saw, and the next step — so you
can pick up exactly where you left off, or get a gentle nudge to finish later.

Built for heavy context-switchers. Detection is just the trigger; the point is taking
the dropped thread off your plate.
